# ResetWell Plus, SEO setup

Primary market: India (with a broader English-speaking audience). Domain: `https://www.resetwellplus.com`.

## What is implemented

- **Per-page meta**: unique `<title>`, `<meta description>` and `<meta keywords>` on every page (see each file in `src/pages` and the `SITE` defaults in `src/data/seo.ts`).
- **Canonical URLs**: emitted on every page from `Astro.site` + the page path.
- **Open Graph + Twitter cards**: title, description, URL, locale (`en_IN`) and a default share image (`public/og-default.svg`). Swap in a 1200x630 PNG for best crawler support.
- **Structured data (JSON-LD)**:
  - `MedicalClinic` (site-wide, in `BaseLayout`).
  - `BreadcrumbList` on every inner page (via `PageLayout`).
  - `FAQPage` on the homepage FAQ and the `/community/faqs` page.
- **Sitemap**: generated by `@astrojs/sitemap` at `/sitemap-index.xml` (the pre-launch `/coming-soon` page is excluded).
- **robots.txt**: `public/robots.txt`, references the sitemap and disallows `/coming-soon`.
- **noindex** on utility/pre-launch pages: `/coming-soon`, `/404`, `/privacy-policy`, `/terms`, and the booking redirect.
- **Locale**: `<html lang="en-IN">`.
